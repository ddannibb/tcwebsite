Current Work


Roadview Project

In our approach, we utilize a domain translation network specifically designed to translate LiDAR data to RGB-D (Red, Green, Blue, and Depth) images. This network is capable of converting LiDAR point cloud data into corresponding RGB-D representations.

Once we obtain the RGB-D images, we leverage them to generate a new set of point clouds.

This process involves extracting depth information from the RGB-D images and converting it back into a point cloud format. By generating these augmented point clouds, we aim to enhance the quality and diversity of the original real-world point cloud dataset.

The primary motivation behind this augmentation is to improve object detection scores. By incorporating the augmented point clouds, which contain additional information derived from the RGB-D images, we expect to enhance the performance of object detection algorithms. The RGB-D data provides richer contextual and spatial details, which can aid in accurately detecting and classifying objects within the scene.

The augmentation process helps to overcome the limitations of relying solely on real-world point cloud data. It allows us to expand the training set with synthetic point clouds generated from the RGB-D images.

This synthetic data introduces variations, such as different lighting conditions, viewpoints, and occlusions, that may not be present in the original real-world data. Consequently, the object detection algorithms trained on this augmented dataset can better generalize to various environmental conditions, leading to improved object detection scores.

In summary, we employ a domain translation network to convert LiDAR data to RGB-D images, and then leverage the RGB-D images to generate augmented point clouds. By incorporating these augmented point clouds into the training process, we aim to enhance object detection scores by providing additional contextual and spatial information.

This approach allows us to overcome the limitations of relying solely on real-world point cloud data and improves the performance of object detection algorithms in various scenarios.
